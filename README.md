# Juego de ahorcado con java!

Una simple practica de un juego del Ahorcado para java donde se intenta adivinar la palabra secreta haciendo intentos con una letra la vez. Se debe adivinar la palabra en no mas de 10 intentos, sino se pierde el juego.

# Heramientas utilizadas:

1. Clase Scanner
2. Declaraciones y asignaciones
3. Arreglos
4. Estructura de control con bucles
5. Clase String y sus metodos valueOf, Lengh y charAt
6. Clase Character y sus metodos
